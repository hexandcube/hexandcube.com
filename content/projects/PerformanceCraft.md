# PerformanceCraft
### A pure performance fabric modpack

Designed to be used as an alternative to Optifine. Increased performance and smoother gameplay, with additional tweaks, fixes, and shader support, designed to work on all Vanilla multiplayer servers.

This modpack is essentially a stripped down version of [Hexandcube's Client Tweaks](/projects/HexandcubesClientTweaks), with all features that could potentially affect gameplay removed, created specifically for playing on public multiplayer servers.

###### Disclaimer
While PerformanceCraft was designed to be used on vanilla Minecraft servers, some of the features (like the zoom key or the minimap) may offer advantages over unmodded clients. Use at your own discretion, and always check with server rules before using any client modifications. 

### Features:

- Improved rendering performance
- Network stack optimizations
- Improved lighting engine
- Improved block and entity culling
- Lowered system performance impact when minimized
- Faster GUI rendering
- Faster animations
- Client and Server side bug fixes
- Better RAM management
- Faster loading time
- Borderless Fullscreen
- Faster block entities
- Fast language switching
- Faster XP orbs
- Shader support
- Zoom key
- Optifine cape and custom cape support
