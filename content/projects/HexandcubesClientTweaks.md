# Hexandcube's Client Tweaks

**Hexandcube's Client Tweaks** is a manually curated collection of **quality-of-life client-side mods** for Minecraft: Java Edition.
The modpack is focused on providing a variety of useful tweaks, features, optimizations, and UI improvements while maintaining 
full compatibility with vanilla Minecraft servers. 

Some of the mods included in this pack offer additional tweaks like performance optimizations server-side for singleplayer mode,
without changing the core vanilla gameplay experience.

###### Disclaimer
While HCT was designed to be used on vanilla Minecraft servers, it was made primarly with private servers in mind.
Some of the features (like block placement tweaks) may offer advantages over unmodded clients, and might trigger anti-cheat systems on public servers.
Use at your own discretion, and always check with server rules before using any client modifications. 

#### Features 

- GUI Improvements
- Saturation and hunger HUD (AppleSkin)
- Shulkerbox and Map Tooltips
- Item Stitching fixes
- Minimap with waypoints
- Crafting and inventory tweaks (with sorting)
- Sound Muffler
- Light level Overlay (F7)
- Comparing equipment
- 3D skin layers
- Enchantment descriptions
- Dynamic lighting
- Performance improvements (Sodium)
- Custom shader support (Iris)
- Optifine and MinecraftCapes.net Cape support
- and more

#### Modpack formats

Since version 4.3 (1.19.4) HCT is published exclusively in the Modrinth modpack format (.mrpack), 
which can be used in any modpack launcher that supports it (like Prism Launcher, MultiMC, GDLauncher, etc)
besides the official Modrinth app. The mods are downloaded from Modrinth's servers when using the Modrinth modpack format.

Versions prior to 4.3 are published in both the Modrinth modpack format and the legacy CurseForge modpack format
(a .zip file with mods, configs, and a manifest.json). When using the CurseForge format version, the launcher will download
mods from CurseForge's servers. The modrinth format versions of the those legacy releases have the mods embedded in the .mrpack file.

Because of the licensing of some of the mods included in the pack, legacy versions 1.0, 1.1, 3.0, 3.1, 3.2, 4.2 beta and 4.2
are not available on Modrinth, but can still be downloaded in the Modrinth modpack here (from the Floofware FS server), or in the legacy
CurseForge format from CurseForge.
